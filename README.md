## ⚰️ Protocolos de Gestão: Wandinha PM (Implantação Hospitalar)

Este projeto apresenta a reestruturação técnica dos modos de interação de IA (Ask, Plan, Edit, Agent e Study) adaptados para a complexa e, muitas vezes, caótica realidade de uma **Gerente de Projetos de TI em Saúde**. 

A solução abandona a assistência genérica em favor de uma personalidade baseada em **Wandinha Addams**: fria, resolutiva, prática e focada na eficiência absoluta. O objetivo é transformar o Copiloto em um braço direito capaz de realizar "autópsias" de erros de sistema, orquestrar cronogramas de implantação hospitalar e executar relatórios técnicos com precisão cirúrgica.

### 🧪 O que foi implementado:
* **Modo ASK (Diagnóstico):** Configurado para identificar falhas em módulos de sistemas como **CELK Saúde e G-VIS**, além de interpretar indicadores do **SUS (Previne Brasil)**.
* **Modo PLAN (Estratégia):** Estruturado para o planejamento *end-to-end* de implantações, mapeamento de requisitos e rituais de capacitação presencial para profissionais de saúde.
* **Modo AGENT (Execução):** Adaptado para a geração autônoma de relatórios diários de atividades, e-mails de alinhamento com stakeholders e documentos técnicos de configuração.
* **Modo STUDY (Mentoria):** Focado no aprendizado acelerado de novas normas de faturamento e funcionalidades de software sem perda de tempo com didáticas tradicionais.

**Tecnologias envolvidas:** Python, TypeScript e integração de sistemas legados de saúde via JSON.
